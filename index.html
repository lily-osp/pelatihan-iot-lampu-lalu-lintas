üåê Fundamental IoT (Internet of Things)

Apa itu IoT?

Internet of Things (IoT) adalah ekosistem perangkat fisik yang tertanam dengan sensor, actuator, software, dan teknologi konektivitas untuk mengumpulkan, mengirim, dan bertukar data dengan perangkat dan sistem lain melalui internet. IoT memungkinkan objek sehari-hari menjadi "cerdas" dan dapat berkomunikasi secara otomatis.

Komponen IoT

  • Sensors - Mengumpulkan data dari lingkungan
  • Actuators - Melakukan aksi berdasarkan data
  • Connectivity - WiFi, Bluetooth, LoRa, dll
  • Processing - Mikrokontroler/komputer
  • Cloud Platform - Penyimpanan & analisis data

Protokol Komunikasi

  • MQTT - Lightweight, ideal untuk IoT
  • HTTP/HTTPS - Web-based communication
  • CoAP - Constrained Application Protocol
  • WebSocket - Real-time bidirectional

Arsitektur IoT

Lapisan IoT (IoT Layers)

Layer 1: Perception

Sensors, actuators, devices

Layer 2: Network

Communication protocols

Layer 3: Application

User interfaces, analytics

üîß Mikrokontroler & ESP32

Apa itu Mikrokontroler?

Mikrokontroler adalah sistem komputer mini yang terintegrasi dalam satu chip (System on Chip - SoC). Berisi CPU, memori (RAM dan ROM), dan berbagai peripheral I/O yang diperlukan untuk mengontrol perangkat elektronik secara real-time. Mikrokontroler dirancang untuk aplikasi embedded yang membutuhkan kontrol langsung terhadap hardware.

Komponen Mikrokontroler

  • CPU - Central Processing Unit
  • Memory - RAM & ROM
  • I/O Ports - Input/Output interfaces
  • Timers - Time-based operations
  • ADC/DAC - Analog conversion

Keunggulan Mikrokontroler

  • Low Power - Konsumsi daya minimal
  • Real-time - Respons cepat
  • Cost-effective - Harga terjangkau
  • Reliable - Stabil untuk embedded systems

ESP32 - Mikrokontroler IoT

Spesifikasi ESP32

  • Dual-core CPU - 2 CPU cores (240MHz) dengan arsitektur Xtensa LX6
  • WiFi & Bluetooth - IEEE 802.11 b/g/n dan Bluetooth 4.2
  • 34 GPIO pins - Interface I/O yang fleksibel dan programmable
  • 520KB SRAM - Memory internal untuk data dan stack
  • 4MB Flash - Penyimpanan program dan data non-volatile
  • Security - Hardware encryption dan secure boot

Peripheral ESP32

  • ADC - 18 channel analog input
  • DAC - 2 channel analog output
  • UART - Serial communication
  • SPI/I2C - Interface sensors
  • PWM - Pulse width modulation

Programming ESP32

Development Environment

  • Arduino IDE - User-friendly interface
  • ESP-IDF - Official framework
  • PlatformIO - Advanced IDE
  • MicroPython - Python programming

Libraries & Tools

  • WiFi - Network connectivity
  • MQTT - IoT communication
  • JSON - Data formatting
  • HTTP Client - Web requests

üîç Troubleshooting & Debugging

Masalah Umum & Solusi

Hardware Issues

  • LED tidak menyala - Cek resistor dan koneksi
  • ESP32 tidak terdeteksi - Cek driver USB dan kabel
  • Upload gagal - Tekan BOOT button saat upload
  • Pin tidak merespons - Hindari pin boot (GPIO 0, 2)

Software Issues

  • WiFi tidak connect - Cek SSID dan password
  • MQTT timeout - Cek kredensial Adafruit IO
  • Memory error - Optimasi penggunaan RAM
  • Watchdog reset - Tambahkan delay() di loop

Debugging Tools

Serial Monitor

  • Baud rate: 115200
  • Debug messages
  • Variable monitoring
  • Error reporting

Testing Methods

  • Unit testing - Test fungsi individual
  • Integration testing - Test sistem lengkap
  • Stress testing - Test performa
  • User acceptance - Test user experience
